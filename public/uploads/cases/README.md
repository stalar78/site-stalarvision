## Case Images

Put local preview images for portfolio cards into this folder.

Expected filenames:

- `case-01-crm-real-estate.jpg`
- `case-02-edtech-platform.jpg`
- `case-03-fintech-landing.jpg`

Notes:

- These files are used by `src/data/cases.ts`.
- Recommended format: `.jpg` or `.webp`.
- Recommended size: at least `1200x900`.
- If a file is missing, the fallback in `src/components/Portfolio.tsx` will be shown.
